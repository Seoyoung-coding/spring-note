GetMapping : "보여주세요" (Read)
용도: 단순히 페이지를 열거나, 데이터를 조회할 때 씁니다.

특징: 주소창에 주소를 직접 치고 들어가는 건 100% GET 방식입니다.

우리 코드 예시:

/board/list: "글 목록 좀 보여줘"

/board/write: "글 쓸 수 있는 빈 화면 좀 보여줘"

🔴 PostMapping : "처리해주세요" (Create/Update/Delete)
용도: 서버의 데이터를 바꾸거나 저장할 때 씁니다. (비밀번호 같은 중요한 데이터도 포함)

특징: 주소창에 보이지 않게 데이터를 숨겨서 보냅니다. 보통 <form> 태그와 함께 쓰입니다.

우리 코드 예시:

/board/write (POST): "내가 쓴 이 글 좀 DB에 저장해줘"



1. @OneToMany 일대다 관계 : 한 엔티티가 여러 엔티티를 참조함
2. 여러 개의 질문(Many)은 한 명의 사용자(One)에게 속합니다.
